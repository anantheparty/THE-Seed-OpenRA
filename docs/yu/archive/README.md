# Archived Yu Docs

This directory holds historical audit notes, live-test traces, and one-off
slice reports that are no longer part of the active execution surface.

Top-level `docs/yu/` should stay focused on:
- current plan/progress
- active execution board / roadmap
- still-referenced architecture or remaining-work summaries

Files moved here are preserved for lookup and postmortem value, but should not
be treated as the current source of truth for runtime behavior or next slices.
